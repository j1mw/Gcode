## Usage ##
```
M28 aaaaaaa
```

### Parameters ###
+ Filename


## Example ##

+ M28 filename.gco

File specified by filename.gco is created (or overwritten if it exists) on the SD card and all subsequent commands sent to the machine are written to that file.
