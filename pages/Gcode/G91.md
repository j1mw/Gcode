## Usage ##

| `G91` | 

### Parameters ###
+ none

## Example ## 
+ `G91`

All coordinates from now on are relative to the last position.

Note: RepRapFirmware uses M83 to set the extruder to relative mode: extrusion is NOT set to relative using G91
