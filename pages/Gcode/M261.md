Deprecated in RRF 2.02 and later. Use M260 instead.
{{site.data.alerts.end}}


## Usage ##

| `M261 Ann Bnn` |

### Parameters ###

+ `Ann` I2C address
+ `Bnn` How many bytes to request

## Example ##

+ `M261 A99 B5` Request 5 bytes from Address 99
