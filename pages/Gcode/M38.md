## Usage ##

| `M38 Filename ` | 

## Paramaters ##
+ `Filename`

## Example ##

+ `M38 gcodes/myfile.g`

Used to compute a hash of a file on the SD card and returns a hexadecimal string which is the SHA1 of the file. If the file cannot be found, then the string "Cannot find file" is returned instead.
