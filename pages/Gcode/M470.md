## Usage ##
```
M470 P"name"
```

### Parameters ###

+ P"name" Name of directory to create

## Example ##

+ M470 P"/sys/config.d"

This will create a new directory on the SD-Card. If not otherwise specified the default root should be the first/internal SD-Card.

Supported in RRF >= 2.03.
