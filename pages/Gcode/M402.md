## Usage ##
```
M402 Pn
```

### Parameters ###

+ P Probe number (RRF 3.01RC4 and later)

## Examples ##

+ M402
+ M402 P1

This runs macro file sys/retractprobe#.g (where # is the probe number) if it exists, otherwise sys/retractprobe.g if it exists.
