## Usage ##

| `M23 "aaaaaaa"` | 

### Parameters ###
+ Filename

## Example ##

+ `M23 filename.gco`

The file specified as filename.gco is selected ready for printing. RepRapFirmware supports long filenames as well as 8.3 format.
