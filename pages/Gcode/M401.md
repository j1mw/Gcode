## Usage ##
```
M401 Pn
```

## Parameters ##

+ P Probe number (RRF 3.01RC4 and later)

## Examples ##

+ M401
+ M401 P1

This runs macro file sys/deployprobe#.g (where # is the probe number) if it exists, otherwise sys/deployprobe.g if it exists.
